##flexbox의 구성

- flexbox는 자식요소인 flex item과 부모 요소인 flex container로 구성된다.

##flexbox 만드는법

- 정렬하려는 요소의 부모요소에 다음과 같이 선언하면 된다.
  <pre><code>.flex_container{
    display : flex; 
  }</code></pre>

##flex container 속성

#### flex-direction

> ### 속성값
>
> - row
> - row-reverse
> - column
> - column-reverse
